Content that doesn't go into the columns (like a navbar) can go up here.

You can put a form or menu in here and use JavaScript to populate content for the main column.

If you don't want this column to scroll, you can set overflow: hidden; on it.

If you want your other columns to be scrollable, set overflow-y: auto; on them. This is how you can customize which columns scroll and which don't!
